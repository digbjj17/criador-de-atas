## 3. Logotipo

- Arquivo: `assets/logo_tecsa_color.png` (versão colorida, preferencial)
- Posição no cabeçalho: canto superior esquerdo
- Altura no cabeçalho: 10mm
- Área de respiro: metade da altura da tipologia ao redor
- Nunca distorcer, alterar cores ou adicionar efeitos

## 4. Layout da Ata

- Formato: A4 (210mm x 297mm)
- Margens: 25mm (superior/inferior), 20mm (esquerda/direita)
- Espaçamento entre linhas: 1.4
- Alinhamento do texto: justificado
- Cabeçalho: logotipo + linha cinza separadora (todas as páginas)
- Rodapé: "TECSA — Confidencial" (esquerda) + "Página X de Y" (direita) + linha cinza acima
